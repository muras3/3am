Why This Happened
Stripe is throttling requests at 847/min. Because checkout-service has no circuit breaker, every checkout attempt still hits Stripe — flooding a saturated dependency. Setting STRIPE_CIRCUIT_OPEN stops new requests from reaching Stripe immediately, letting the rate limit window reset and orders recover within 2–5 minutes.
