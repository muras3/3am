v1 → v2 Changes

v1 Issues

  • 全要素間 gap: 24px で均一 — リズムなし
  • サービスと依存が flat flex で同格
  • コネクタ矢印が flex item として挟まれて弱い
  • 全カードが border-radius: 6px — 形で区別不可
  • Critical route が dot の色でしか分からない
  • サービス名 12px = 依存名 12px — 主従の差なし

v2 Fixes

  • 3ゾーン分離 — サービス / edge / 依存で空間的階層
  • サービスが主（左、大）、依存が従（右、小）
  • SVG edge を独立ゾーンに — animated dots 付き
  • 依存 = pill (border-radius: 24px) — 形で区別
  • Critical route = 左アクセント線 + 背景 tint
  • サービス名 13px > 依存名 12px — 主従明確
